## Installation

```bash
npm install
```

## Running the app

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```

## Seeder

```bash
# Drop all data
npm run seed:drop

# create seeder
npm run test:e2e

# refresh seeder
npm run seed:drop && npm run test:e2e
```
